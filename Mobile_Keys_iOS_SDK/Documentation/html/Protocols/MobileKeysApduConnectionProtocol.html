MobileKeysApduConnectionProtocol

@protocol MobileKeysApduConnectionProtocol <NSObject>

Interface for sending APDU commands to, and receiving APDU responses from a Seos card.

How implementations of this interface would look depends on where the API is used, for instance the local file system, Bluetooth, NFC, USB or WIFI, etc.
