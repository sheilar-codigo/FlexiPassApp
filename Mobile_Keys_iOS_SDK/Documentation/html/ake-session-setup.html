AKE Session Setup

Using AKE to authenticate to a card requires an active session to the Terminal. The following example code sets up basic privacy keys and a session to the Terminal using the MobileKeysSeosProvider API. A second MobileKeysSeosProvider is then required to create the APDU connection session to the Card.

The code has been tested and works using the example keys below the code example.

Overview of the code

  1. Setup privacy keys using the test vectors (same as in the json keys below)
  2. Open a session to the Mobile Access SDK default MobileKeysSeosProvider
    • This instance of Seos is expected to have the “Terminal side test key”
  3. Setup the custom Card Connection (not provided by the example code)
    • This instance of Seos is expected to have the “Card side test key”
  4. Establish a connection to the Card by utilizing the already connected Terminal Connection MobileKeysSeosProvider
  5. Read Tag FF00 from the Card
  6. Close the connection

Objective C example code for setting up a Card session

- (void) doAke{

    // Privacy keys for Terminal side (using test vectors)
    MobileKeysSymmetricKeyPair * terminalPrivacyKey= [[MobileKeysSymmetricKeyPair alloc] initWithMacKey:[self hexToData:@"8aa83bf8cbda10620bc1bf19fbb6cd58"] encKey:[self hexToData:@"5A5A5A5A5A5A5A5A5A5A5A5A5A5A5A5A"] encAlg:MOBILE_KEYS_ENCRYPTION_ALGO_AES_128];
    MobileKeysPrivacyKeySet * terminalPrivacyKeySet = [[MobileKeysPrivacyKeySet alloc] initWithKeyNumber:MOBILE_KEYS_KEY_0 symmetricKeyPair:terminalPrivacyKey encryptionAlgorithm:MOBILE_KEYS_ENCRYPTION_ALGO_AES_128];

    //Terminal Session Parameters
    MobileKeysSessionParameters *sessionParametersNoAuthTerminal = [[MobileKeysSessionParameters alloc] initWithMobileKeysSelectionType:MobileKeysSelectionADF mobileKeysPrivacyKeySet:terminalPrivacyKeySet mobileKeysAuthenticationKeySet:nil];
    [sessionParametersNoAuthTerminal setRequireSelectAid:YES];
    [sessionParametersNoAuthTerminal setSingleOidSelection:[self hexToData:@"0202020202020202"]];
    [sessionParametersNoAuthTerminal setAutoSplitLargeApduCommands:YES];
    [sessionParametersNoAuthTerminal setAutoJoinMultipleApduResponses:YES];
    [sessionParametersNoAuthTerminal setUseTestVectors:NO];

    // Privacy keys for Card side (using test vectors)
    MobileKeysSymmetricKeyPair * cardPrivacyKey= [[MobileKeysSymmetricKeyPair alloc] initWithMacKey:[self hexToData:@"8aa83bf8cbda10620bc1bf19fbb6cd58"] encKey:[self hexToData:@"5A5A5A5A5A5A5A5A5A5A5A5A5A5A5A5A"] encAlg:MOBILE_KEYS_ENCRYPTION_ALGO_AES_128];
    MobileKeysPrivacyKeySet * cardPrivacyKeySet = [[MobileKeysPrivacyKeySet alloc] initWithKeyNumber:MOBILE_KEYS_KEY_0 symmetricKeyPair:cardPrivacyKey encryptionAlgorithm:MOBILE_KEYS_ENCRYPTION_ALGO_AES_128];

    // Card Session Parameters
    MobileKeysSessionParameters *sessionParametersNoAuthCard = [[MobileKeysSessionParameters alloc] initWithMobileKeysSelectionType:MobileKeysSelectionADF mobileKeysPrivacyKeySet:cardPrivacyKeySet mobileKeysAuthenticationKeySet:nil];
    [sessionParametersNoAuthCard setRequireSelectAid:YES];
    [sessionParametersNoAuthCard setSingleOidSelection:[self hexToData:@"0202020202020202"]];
    [sessionParametersNoAuthCard setAutoSplitLargeApduCommands:YES];
    [sessionParametersNoAuthCard setAutoJoinMultipleApduResponses:YES];
    [sessionParametersNoAuthCard setUseTestVectors:NO];

    // Get the default Mobile Access SDK Seos Provider
     MobileKeysSeosProvider *mobileKeysSeosProviderTerminal = [_mobileKeysManager defaultMobileKeysSeosProvider];

    NSError *error = nil;
    // Open a session to the Terminal side
    MobileKeysSeosSession * terminalSession=  [mobileKeysSeosProviderTerminal establishSessionWithParams:sessionParametersNoAuthTerminal description:@"Terminal session" withError:&error];
    NSData  * selectedOid = [terminalSession currentlySelectedOid];
    // NOTICE: Validate that selected Oid is correct and error is nil

    // Get a reference to the cardConnection
    // NOTICE: Create your own apdu connection to the card (integrator specific)
    id <MobileKeysApduConnectionProtocol> cardApduConnection;

    // Instantiate a secondary MobileKeysSeosProvider with the cardConnection
    MobileKeysSeosProvider *mobileKeysSeosProviderCard = [[MobileKeysSeosProvider alloc] initWithConnection:_cardApduConnection];

    // Open a session to the Card by providing the terminal side MobileKeysSeosProvider and all other required information
    MobileKeysSeosSession *cardSession = [mobileKeysSeosProviderCard establishCardSessionUsingTerminalSession:terminalSession cardSideParameters:sessionParametersNoAuthCard cardKeyNumber:MOBILE_KEYS_KEY_4 terminalKeyNumber:MOBILE_KEYS_KEY_4 withError:&error];
    // NOTICE: We currently return a session object, but it is recommended that you keep track of your MobileKeysSeosProvider for the time being.

    // Setup a small test
    MobileKeysSeosTag *tagFF00 = [[MobileKeysSeosTag alloc] initWithTagNumber:0xFF00];
    MobileKeysGetDataApduCommand *getData = [[MobileKeysGetDataApduCommand alloc] initWithTag:tagFF00 secureMessaging:YES];

    // use the new MobileKeysSeosProvider instance to talk to the card
    MobileKeysApduResponse *responseRead = [cardSession processApduCommand:getData withError:&error];
    NSLog(@"Read tag 0xFF00. Data = %@", responseRead.data);
    NSLog(@"GetData returned StatusWord %@", responseRead.statusWordData);
    // NOTICE: Should return status 9000

    // Close all sessions when done
    [cardSession closeSessionWithError:&error];
    [terminalSession closeSessionWithError:&error];
}



- (NSData *)hexToData:(NSString *)hex {
    NSMutableData *data = [[NSMutableData alloc] init];
    NSUInteger idx;
    for (idx = 0; idx + 2 <= hex.length; idx += 2) {
        NSRange range = NSMakeRange(idx, 2);
        NSString *hexStr = [hex substringWithRange:range];
        NSScanner *scanner = [NSScanner scannerWithString:hexStr];
        unsigned int intValue;
        [scanner scanHexInt:&intValue];
        char bytesToAppend[1] = {intValue};
        [data appendBytes:bytesToAppend length:1];
    }
    return data;
}
