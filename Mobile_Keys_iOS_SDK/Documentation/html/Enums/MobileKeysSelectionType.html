MobileKeysSelectionType

enum MobileKeysSelectionType {}

Describes the type of ADF selection to use during session establishment.

  • MobileKeysSelectionADF

    Generates a command to select an ADF as specified by the SELECT ADF by sequence of OID with Privacy.

    Declaration

    Objective-C

    MobileKeysSelectionADF
  • MobileKeysSelectionGDF

    Selects the GDF.

    Declaration

    Objective-C

    MobileKeysSelectionGDF
  • MobileKeysSelectionExtendedADF

    Generates a command to select an ADF. This version of select ignores the disabled state of an ADF, and can be used to select an ADF even if it’s disabled.

    Note

    This version of select ADF requires previous authentication with a key granting “core administration” permissions.

    Declaration

    Objective-C

    MobileKeysSelectionExtendedADF
  • MobileKeysSelectionEmptySeos

    Selects an empty Seos card, use the {@link GenesisPrivacyKeyset} for privacy when using this selection mode.

    Declaration

    Objective-C

    MobileKeysSelectionEmptySeos
  • MobileKeysSelectionNone

    Do not perform any selection.
